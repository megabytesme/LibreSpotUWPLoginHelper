**LibreSpotUWP Login Helper** is a helper utility which allows for easy sign in for the LibreSpotUWP app, using a QR code. Meant for older devices which can't sign in with Spotify directly.

**Disclaimer:** LibreSpotUWP is an unofficial, third-party implementation of a Spotify client for the Universal Windows Platform (UWP). This project is not affiliated with, endorsed, or sponsored by Spotify AB. Spotify is a trademark of Spotify AB.
